Foot Traffic Index
Foot traffic data was sourced from GroundTruth's Daily Foot Traffic Tracker [groundtruth.com/insight/industry-foot-traffic-tracker]. Data is available from December 20, 2019 through March 3, 2021. GroundTruth data is indexed to show relative increases and decreases in daily activity at places of interest.
Daily Covid Cases and Vaccinations
Covid case numbers are from IDPH [observablehq.com/@chicagoreporter/historical-illinois-covid-19-zip-code-data]. The plot shows the change in number of confirmed covid cases averaged over the previous seven days. The vaccination data is from the Chicago Data Portal API [data.cityofchicago.org]. The plot shows the number of vaccine doses administered averaged over the previous seven days.
Demographic Data
Demographic data is from the Census API for the American Community Survey for 2009-2019. [www.census.gov/data/developers/data-sets/acs-5year.html].


From public health impacts to economic downturn, the COVID-19 pandemic - and requisite social distancing measures - have altered the lives of many Chicago residents. Using data from a variety of sources at the ZIP Code level, we built a dashboard that visualizes some of the changes in the daily lives of Chicagoans over the course of the pandemic. This project is primarily focused on observing the ways in which daily activities have and have not begun to return to their pre-pandemic forms in different parts of the city.

Explore the data by selecting a ZIP Code region in the map.
